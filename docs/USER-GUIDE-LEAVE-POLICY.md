# คู่มือผู้ใช้: สิทธิ์วันลา

## ทำไมส่งคำขอลาไม่ได้

ถ้าระบบแจ้งว่าไม่สามารถส่งคำขอลาได้ อาจเกิดจาก:

- ยังไม่ได้กำหนดประเภทพนักงาน
- ยังไม่ได้กำหนดวันที่เริ่มปฏิบัติงาน
- อายุงานยังไม่ครบตาม policy
- วันลาคงเหลือไม่เพียงพอ
- มีคำขอรออนุมัติที่กันยอดไว้แล้ว

## ยอดวันลาคงเหลือ

ระบบคำนวณจาก:

```text
คงเหลือใช้ได้ = สิทธิ์ประจำปี + ยกมาจากปีก่อน + ปรับปรุง - ใช้ไปแล้ว - รออนุมัติ
```

คำขอที่ยังรออนุมัติจะถูกนับเป็น `รออนุมัติ` เพื่อป้องกันการส่งคำขอหลายใบเกินยอดจริง

## หน้าสร้างคำขอลา

เมื่อเลือกประเภทลาและวันที่ ระบบจะแสดง:

- ประเภทบุคลากร
- ปีงบประมาณ
- สิทธิ์ประจำปี
- ใช้ไปแล้ว
- รออนุมัติ
- คงเหลือใช้ได้
- จำนวนวันที่คำขอนี้ใช้
- warning/note จาก policy

ถ้าไม่ผ่าน policy ปุ่มบันทึกจะถูกปิด และ backend จะตรวจซ้ำเสมอ

## ผู้ดูแลระบบต้องตรวจอะไร

ก่อนให้ผู้ใช้เริ่มส่งคำขอลา ควรตรวจ:

- ผู้ใช้มีประเภทพนักงานถูกต้อง
- ผู้ใช้มีวันที่เริ่มปฏิบัติงาน
- ผู้ใช้มี balance ของปีงบประมาณปัจจุบัน
- ประเภทลาที่ใช้งานมี policy rule active
