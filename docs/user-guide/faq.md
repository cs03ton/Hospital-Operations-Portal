# ❓ FAQ คำถามที่พบบ่อย

รวมคำถามที่ผู้ใช้งาน HOP Phase 1 พบบ่อย พร้อมแนวทางตรวจสอบเบื้องต้น

## 📲 LINE ไม่แจ้งเตือนทำอย่างไร

1. ตรวจว่าเชื่อมต่อ LINE แล้วในหน้า `ข้อมูลส่วนตัวของฉัน`
2. ตรวจว่าเพิ่มเพื่อน LINE OA แล้ว
3. ตรวจว่า LINE User ID ไม่ซ้ำกับบัญชีอื่น
4. หากยังไม่ได้รับแจ้งเตือน ให้ติดต่อผู้ดูแลระบบ

> 💡 **Tip:** ถ้าเพิ่งเชื่อมต่อ LINE ให้ลองกดส่งข้อความทดสอบถึงตัวเองก่อน

## 🔐 ลืมรหัสผ่านทำอย่างไร

ติดต่อผู้ดูแลระบบหรือเจ้าหน้าที่ IT เพื่อรีเซ็ตรหัสผ่านตามขั้นตอนของโรงพยาบาล

> ⚠️ **ห้ามทำ:** ห้ามใช้รหัสผ่านร่วมกับผู้อื่น หรือส่งรหัสผ่านผ่านช่องทางสาธารณะ

## 🔑 เปลี่ยนรหัสผ่านเองได้ไหม

ได้ ผู้ใช้ที่ Login แล้วสามารถเปลี่ยนรหัสผ่านของตนเองได้จากเมนูผู้ใช้งานมุมขวาบน

1. คลิกชื่อผู้ใช้งานหรือรูปโปรไฟล์
2. เลือก `เปลี่ยนรหัสผ่าน`
3. กรอกรหัสผ่านปัจจุบัน
4. ตั้งรหัสผ่านใหม่ให้ผ่าน Password Policy
5. กด `เปลี่ยนรหัสผ่าน`
6. Login ใหม่หลังระบบออกจากระบบอัตโนมัติ

> 💡 **หมายเหตุ:** หากจำรหัสผ่านปัจจุบันไม่ได้ ต้องให้ Admin reset password ให้

## 🚪 ทำไมเปลี่ยนรหัสผ่านแล้วถูกพาไปหน้า Login

เป็นพฤติกรรมปกติของระบบ หลังเปลี่ยนรหัสผ่านสำเร็จ ระบบจะยกเลิก session เดิมทั้งหมดเพื่อความปลอดภัย จากนั้นให้ Login ใหม่ด้วยรหัสผ่านใหม่

## 🧩 รหัสผ่านใหม่ต้องมีเงื่อนไขอะไรบ้าง

โดยทั่วไปต้องมี:

- ความยาวอย่างน้อย 8 ตัวอักษร
- ตัวพิมพ์ใหญ่
- ตัวพิมพ์เล็ก
- ตัวเลข
- อักขระพิเศษ
- ไม่ใช้ username เป็นส่วนหนึ่งของรหัสผ่าน
- ไม่ซ้ำกับรหัสผ่านเดิม

## 💼 ขอวันลาเกินสิทธิ์ทำไมไม่ได้

ระบบตรวจยอดวันลาคงเหลือ ประเภทลา คำขอที่รออนุมัติ และเงื่อนไขตามประเภทบุคลากร หากยอดไม่พอ ระบบจะไม่ให้ส่งคำขอ

## ✅ ทำไมไม่เห็นปุ่มอนุมัติ

ปุ่มอนุมัติจะแสดงเฉพาะผู้อนุมัติที่ถึงคิวปัจจุบันเท่านั้น หากไม่ถึงคิวหรือไม่มีสิทธิ์ ระบบจะไม่แสดงปุ่ม

## 🟠 สถานะ “ตีกลับรอแก้ไข” คืออะไร

หมายถึงผู้อนุมัติยังไม่ปฏิเสธคำขอ แต่ต้องการให้ผู้ขอแก้ไขข้อมูลหรือไฟล์แนบก่อน

1. เปิดรายละเอียดคำขอลา
2. อ่านเหตุผลที่ผู้อนุมัติระบุ
3. แก้ไขข้อมูลหรือไฟล์แนบตามคำแนะนำ
4. กด `ส่งคำขอใหม่`

> 💡 **จำง่าย:** สถานะนี้ยังไม่ใช่ `ไม่อนุมัติ` จึงสามารถแก้ไขและส่งกลับเข้ากระบวนการอนุมัติได้

## 📎 ดูตัวอย่างไฟล์แนบไม่ได้ทำอย่างไร

1. ตรวจว่าไฟล์เป็น PDF, JPG, JPEG, PNG หรือ WEBP
2. หากระบบขึ้นข้อความ `ไม่รองรับการแสดงตัวอย่างไฟล์ประเภทนี้` ให้ใช้ปุ่ม `ดาวน์โหลด` ตามสิทธิ์
3. หากไฟล์ที่รองรับยังเปิดไม่ได้ ให้แจ้ง IT พร้อมเลขที่คำขอ ชื่อไฟล์ และเวลาที่พบปัญหา

## 🔄 เปลี่ยนบัญชี LINE ได้ไหม

ได้ ให้เข้า `ข้อมูลส่วนตัวของฉัน` แล้วกด `ยกเลิกการเชื่อมต่อ LINE` จากนั้นสร้างรหัสเชื่อมต่อใหม่และผูกบัญชี LINE ใหม่

## 📄 PDF เปิดไม่ได้ทำอย่างไร

1. ลองดาวน์โหลดใหม่อีกครั้ง
2. ตรวจว่า browser ไม่ได้ block download
3. หากยังเปิดไม่ได้ ให้แจ้งเลขที่คำขอและเวลาเกิดปัญหาแก่ผู้ดูแลระบบ

## 🧭 ไม่เห็นเมนูที่ต้องใช้งาน

1. Logout แล้ว Login ใหม่
2. ตรวจสอบ role ของตนเอง
3. ติดต่อผู้ดูแลระบบเพื่อตรวจ permission

## 📅 วันหยุดราชการลาไม่ได้ใช่ไหม

ใช่ ระบบจะไม่อนุญาตให้เลือกวันหยุดราชการเป็นวันลา และในปฏิทินจะแสดงเป็นวันหยุด

## 🧑‍💻 ต้องแจ้ง IT อย่างไรเมื่อพบปัญหา

ควรแจ้งข้อมูลต่อไปนี้:

- ชื่อผู้ใช้
- เมนูหรือหน้าที่พบปัญหา
- เวลาที่เกิดปัญหา
- ข้อความ error ที่แสดง
- เลขที่คำขอ ถ้าเกี่ยวข้องกับระบบลา
- รูปภาพหน้าจอ ถ้ามี

## 💾 Backup Center ขึ้น Warning ต้องทำอย่างไร

สำหรับ Admin หรือ SuperAdmin ให้ตรวจตามลำดับ:

1. เปิด `จัดการระบบ` > `Backup Center`
2. ตรวจว่า database backup ล่าสุดอยู่ที่ `/opt/hop/backups/postgres`
3. ตรวจว่าชื่อไฟล์เป็นรูปแบบ `hopdb_YYYYMMDD_HHMMSS.backup`
4. ตรวจว่า storage backup ล่าสุดอยู่ที่ `/opt/hop/backups/storage`
5. ตรวจ log backup ล่าสุด
6. หาก backup ไม่สำเร็จ ให้แจ้ง IT พร้อมเวลาที่พบปัญหาและข้อความ error

> 💡 **Tip:** Backup Center ใช้สำหรับตรวจสถานะเท่านั้น การ restore production ต้องทำใน maintenance window และมีผู้อนุมัติ
