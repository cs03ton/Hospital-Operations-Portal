# 🧑‍💼 คู่มือหัวหน้าหน่วยงาน

คู่มือนี้สำหรับหัวหน้าหน่วยงานหรือผู้อนุมัติขั้นต้นที่ต้องตรวจคำขอลาของเจ้าหน้าที่ในหน่วยงาน และใช้ข้อมูลการลาเพื่อวางแผนกำลังคน

> 💡 **แนวคิดสำคัญ:** หัวหน้าจะเห็นและอนุมัติเฉพาะคำขอที่ถึงคิวของตนเองเท่านั้น

## 🔑 บัญชีและรหัสผ่าน

หัวหน้าหน่วยงานสามารถเปลี่ยนรหัสผ่านของตนเองได้จากเมนูผู้ใช้งานมุมขวาบน

1. คลิกชื่อหรือรูปโปรไฟล์มุมขวาบน
2. เลือก `เปลี่ยนรหัสผ่าน`
3. กรอกรหัสผ่านปัจจุบัน
4. ตั้งรหัสผ่านใหม่ให้ผ่าน Password Policy
5. กด `เปลี่ยนรหัสผ่าน`
6. Login ใหม่หลังระบบออกจากระบบอัตโนมัติ

> ⚠️ **สำคัญ:** ห้ามให้ผู้อื่นใช้บัญชีหัวหน้าเพื่ออนุมัติคำขอ เพราะทุก action ถูกบันทึกใน Audit Log

## ✅ ดูงานรออนุมัติ

1. ไปที่เมนู `ระบบลา`
2. เลือก `งานรออนุมัติของฉัน`
3. ตรวจสอบรายการคำขอที่ถึงคิวของตนเอง
4. ใช้ตัวกรองเพื่อค้นหาตามสถานะ ประเภทลา หรือชื่อผู้ขอ

> 🔔 **Notification Badge:** ตัวเลขแจ้งเตือนจะแสดงเฉพาะงานที่รอให้หัวหน้าดำเนินการ

## 📊 ใช้งาน Dashboard สำหรับหัวหน้าหน่วยงาน

Dashboard ของหัวหน้าหน่วยงานจะแสดงข้อมูลเรียงจากงานส่วนตัวไปยังงานของทีม เพื่อช่วยให้แยกบทบาท “ผู้ขอลา” และ “หัวหน้าหน่วยงาน” ได้ชัดเจน

ลำดับการอ่านที่แนะนำ:

1. `คำขอลาของฉัน`
2. `คำขอยกเลิกใบลา`
3. `คำขอลาของฉันที่รออนุมัติ`
4. `คำขอลาของหน่วยงาน`
5. งานอนุมัติและสถิติทีม

### 1. คำขอลาของฉันที่รออนุมัติ

ส่วนนี้แสดงคำขอลาของหัวหน้าเองที่ยังอยู่ในสถานะ `รออนุมัติ`

1. เข้าเมนู `Dashboard`
2. เปิด `Dashboard ระบบลา`
3. ดู Card `คำขอลาของฉันที่รออนุมัติ`
4. กด `ดูทั้งหมด` เพื่อเปิดรายการคำขอลาของตนเองที่รออนุมัติ

> 💡 **หมายเหตุ:** รายการที่อนุมัติแล้ว ไม่อนุมัติ ยกเลิก หรือตีกลับรอแก้ไข จะไม่ถูกนับใน Card นี้

### 2. คำขอลาของหน่วยงาน

ส่วนนี้แสดงคำขอลาของเจ้าหน้าที่ในหน่วยงานเดียวกัน โดยไม่รวมคำขอของหัวหน้าเอง

1. เข้าเมนู `Dashboard`
2. เปิด `Dashboard ระบบลา`
3. ดู Card `คำขอลาของหน่วยงาน`
4. กด `ดูทั้งหมด` เพื่อเปิดรายการคำขอลาของหน่วยงาน

ข้อมูลที่แสดงใน Card ประกอบด้วย:

- เลขที่คำขอ
- ชื่อผู้ขอ
- ประเภทลา
- วันที่ลา
- จำนวนวัน
- สถานะ
- ผู้อนุมัติปัจจุบัน

> ✅ **ประโยชน์:** หัวหน้าสามารถเห็นภาพรวมคำขอของทีมได้ โดยไม่ปะปนกับคำขอของตนเอง

## 🔎 ตรวจรายละเอียดคำขอลา

1. คลิกคำขอที่ต้องการตรวจสอบ
2. ตรวจข้อมูลผู้ขอ ประเภทลา วันที่ลา จำนวนวันลา และเหตุผล
3. ตรวจเอกสารแนบถ้ามี โดยกด `ดูตัวอย่าง` ก่อนดาวน์โหลด
4. ตรวจ Timeline การอนุมัติว่าถึงขั้นของตนเองจริง

### ✅ Checklist ผู้อนุมัติ

- [ ] วันที่ลาสอดคล้องกับภาระงานของหน่วยงาน
- [ ] จำนวนวันลาถูกต้อง
- [ ] เหตุผลการลาชัดเจน
- [ ] เอกสารแนบครบถ้วนถ้าประเภทลาต้องใช้
- [ ] ไม่ใช่คำขอของตนเอง
- [ ] Timeline แสดงว่าถึงคิวของตนเอง

## 🟢 อนุมัติคำขอ

1. เปิดรายละเอียดคำขอ
2. ตรวจสอบข้อมูลให้ครบถ้วน
3. ใส่หมายเหตุถ้าต้องการ
4. กด `อนุมัติ`
5. ระบบจะส่งต่อไปยังผู้อนุมัติขั้นถัดไปโดยอัตโนมัติ

> ✅ **หลังอนุมัติ:** Badge ของหัวหน้าจะหาย และคำขอจะไปยังผู้อนุมัติขั้นถัดไป

## 🔴 ไม่อนุมัติคำขอ

1. เปิดรายละเอียดคำขอ
2. ใส่เหตุผลการไม่อนุมัติให้ชัดเจน
3. กด `ไม่อนุมัติ`
4. ระบบจะแจ้งผู้ขอผ่าน Notification และ LINE ถ้ามีการเชื่อมต่อ

> ✍️ **คำแนะนำ:** ควรระบุเหตุผลให้ชัดเจน เช่น เอกสารไม่ครบ วันที่กระทบเวร หรือยอดวันลาไม่สอดคล้อง

## 🟠 ตีกลับรอแก้ไข

ใช้เมื่อคำขอยังไม่ควรถูกปฏิเสธถาวร แต่ต้องให้ผู้ขอแก้ไขข้อมูลหรือแนบเอกสารเพิ่มเติม

1. เปิดรายละเอียดคำขอลา
2. ตรวจสอบข้อมูลและไฟล์แนบผ่านปุ่ม `ดูตัวอย่าง`
3. กด `ตีกลับรอแก้ไข`
4. ระบุเหตุผลให้ชัดเจน เช่น `กรุณาแนบใบรับรองแพทย์` หรือ `กรุณาระบุที่อยู่ระหว่างลา`
5. ยืนยันการตีกลับ

หลังตีกลับแล้ว รายการจะออกจาก `งานรออนุมัติของฉัน` จนกว่าผู้ขอจะแก้ไขและกด `ส่งคำขอใหม่`

> ✅ **ใช้ให้ถูกจังหวะ:** หากต้องการให้ผู้ขอแก้ไขเอกสาร ให้ใช้ `ตีกลับรอแก้ไข` แทน `ไม่อนุมัติ`

## 📲 การแจ้งเตือน LINE

เมื่อมีคำขอถึงคิว หัวหน้าหน่วยงานจะได้รับข้อความ LINE Flex Message พร้อมปุ่ม:

- 👁️ ดูรายละเอียด
- ✅ อนุมัติ
- ❌ ไม่อนุมัติ

> ⚠️ **ข้อควรระวัง:** ให้ตรวจ URL ก่อนดำเนินการจาก LINE และห้ามส่งต่อข้อความอนุมัติให้ผู้อื่น

## 🔁 อนุมัติคำขอยกเลิกใบลา

คำขอยกเลิกใบลาใช้เลขที่เอกสาร `LVC-xxxx` และใช้สายอนุมัติเดียวกับคำขอลาปกติ

1. เปิด `งานรออนุมัติของฉัน` หรือเมนู `คำขอยกเลิกใบลา`
2. เปิดรายละเอียดคำขอยกเลิกใบลา
3. ตรวจใบลาเดิมที่อ้างอิง ประเภทลา วันที่ลา จำนวนวันที่จะคืนยอด และเหตุผล
4. ตรวจสายอนุมัติว่าถึงขั้นของตนเอง
5. เลือก `อนุมัติ`, `ไม่อนุมัติ` หรือ `ตีกลับรอแก้ไข`

> ✅ **หลังอนุมัติครบทุกขั้น:** ระบบจะเปลี่ยนใบลาเดิมเป็น `ยกเลิกหลังอนุมัติ` และคืนยอดวันลาให้อัตโนมัติ

## 📅 ดูปฏิทินการลาในหน่วยงาน

1. ไปที่ `ปฏิทินการลา`
2. เลือกเดือนหรือปีที่ต้องการดู
3. คลิกวันที่เพื่อดูรายละเอียดเจ้าหน้าที่ที่ลา
4. ใช้ข้อมูลเพื่อวางแผนกำลังคนในหน่วยงาน

> 📌 **ใช้งานจริง:** ปฏิทินช่วยให้เห็นภาพรวมว่าวันใดมีเจ้าหน้าที่ลาเยอะ และควรจัดเวรทดแทนหรือไม่
