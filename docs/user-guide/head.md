# 🧑‍💼 คู่มือหัวหน้าหน่วยงาน

คู่มือนี้สำหรับหัวหน้าหน่วยงานหรือผู้อนุมัติขั้นต้นที่ต้องตรวจคำขอลาของเจ้าหน้าที่ในหน่วยงาน และใช้ข้อมูลการลาเพื่อวางแผนกำลังคน

> 💡 **แนวคิดสำคัญ:** หัวหน้าจะเห็นและอนุมัติเฉพาะคำขอที่ถึงคิวของตนเองเท่านั้น

## 🔑 บัญชีและรหัสผ่าน

หัวหน้าหน่วยงานสามารถเปลี่ยนรหัสผ่านของตนเองได้จากเมนูผู้ใช้งานมุมขวาบน

1. คลิกชื่อหรือรูปโปรไฟล์มุมขวาบน
2. เลือก `เปลี่ยนรหัสผ่าน`
3. กรอกรหัสผ่านปัจจุบัน
4. ตั้งรหัสผ่านใหม่ให้ผ่าน Password Policy
5. กด `เปลี่ยนรหัสผ่าน`
6. Login ใหม่หลังระบบออกจากระบบอัตโนมัติ

> ⚠️ **สำคัญ:** ห้ามให้ผู้อื่นใช้บัญชีหัวหน้าเพื่ออนุมัติคำขอ เพราะทุก action ถูกบันทึกใน Audit Log

## ✅ ดูงานรออนุมัติ

1. ไปที่เมนู `ระบบลา`
2. เลือก `งานรออนุมัติของฉัน`
3. ตรวจสอบรายการคำขอที่ถึงคิวของตนเอง
4. ใช้ตัวกรองเพื่อค้นหาตามสถานะ ประเภทลา หรือชื่อผู้ขอ

> 🔔 **Notification Badge:** ตัวเลขแจ้งเตือนจะแสดงเฉพาะงานที่รอให้หัวหน้าดำเนินการ

## 🔎 ตรวจรายละเอียดคำขอลา

1. คลิกคำขอที่ต้องการตรวจสอบ
2. ตรวจข้อมูลผู้ขอ ประเภทลา วันที่ลา จำนวนวันลา และเหตุผล
3. ตรวจเอกสารแนบถ้ามี
4. ตรวจ Timeline การอนุมัติว่าถึงขั้นของตนเองจริง

### ✅ Checklist ผู้อนุมัติ

- [ ] วันที่ลาสอดคล้องกับภาระงานของหน่วยงาน
- [ ] จำนวนวันลาถูกต้อง
- [ ] เหตุผลการลาชัดเจน
- [ ] เอกสารแนบครบถ้วนถ้าประเภทลาต้องใช้
- [ ] ไม่ใช่คำขอของตนเอง
- [ ] Timeline แสดงว่าถึงคิวของตนเอง

## 🟢 อนุมัติคำขอ

1. เปิดรายละเอียดคำขอ
2. ตรวจสอบข้อมูลให้ครบถ้วน
3. ใส่หมายเหตุถ้าต้องการ
4. กด `อนุมัติ`
5. ระบบจะส่งต่อไปยังผู้อนุมัติขั้นถัดไปโดยอัตโนมัติ

> ✅ **หลังอนุมัติ:** Badge ของหัวหน้าจะหาย และคำขอจะไปยังผู้อนุมัติขั้นถัดไป

## 🔴 ไม่อนุมัติคำขอ

1. เปิดรายละเอียดคำขอ
2. ใส่เหตุผลการไม่อนุมัติให้ชัดเจน
3. กด `ไม่อนุมัติ`
4. ระบบจะแจ้งผู้ขอผ่าน Notification และ LINE ถ้ามีการเชื่อมต่อ

> ✍️ **คำแนะนำ:** ควรระบุเหตุผลให้ชัดเจน เช่น เอกสารไม่ครบ วันที่กระทบเวร หรือยอดวันลาไม่สอดคล้อง

## 📲 การแจ้งเตือน LINE

เมื่อมีคำขอถึงคิว หัวหน้าหน่วยงานจะได้รับข้อความ LINE Flex Message พร้อมปุ่ม:

- 👁️ ดูรายละเอียด
- ✅ อนุมัติ
- ❌ ไม่อนุมัติ

> ⚠️ **ข้อควรระวัง:** ให้ตรวจ URL ก่อนดำเนินการจาก LINE และห้ามส่งต่อข้อความอนุมัติให้ผู้อื่น

## 📅 ดูปฏิทินการลาในหน่วยงาน

1. ไปที่ `ปฏิทินการลา`
2. เลือกเดือนหรือปีที่ต้องการดู
3. คลิกวันที่เพื่อดูรายละเอียดเจ้าหน้าที่ที่ลา
4. ใช้ข้อมูลเพื่อวางแผนกำลังคนในหน่วยงาน

> 📌 **ใช้งานจริง:** ปฏิทินช่วยให้เห็นภาพรวมว่าวันใดมีเจ้าหน้าที่ลาเยอะ และควรจัดเวรทดแทนหรือไม่
