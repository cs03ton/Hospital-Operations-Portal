# 👤 คู่มือผู้ใช้งานทั่วไป

คู่มือนี้สำหรับเจ้าหน้าที่โรงพยาบาลนาหมื่นที่ใช้งาน Hospital Operations Portal (HOP) Phase 1 เพื่อขอลา ติดตามสถานะ ดาวน์โหลดเอกสาร และรับการแจ้งเตือนผ่าน LINE Official Account

> 💡 **ภาพรวมเร็ว:** เข้าสู่ระบบ → ตรวจข้อมูลส่วนตัว → สร้างคำขอลา → แนบไฟล์ → ส่งอนุมัติ → ติดตามสถานะ → ดาวน์โหลด PDF

## 🔐 เข้าสู่ระบบ

1. เปิดเว็บไซต์ HOP ตาม URL ที่โรงพยาบาลกำหนด
2. กรอก `Username` และ `Password`
3. กดปุ่ม `เข้าสู่ระบบ`
4. ตรวจสอบชื่อผู้ใช้งานที่มุมขวาบน

> ⚠️ **ข้อควรระวัง:** ห้ามส่งต่อรหัสผ่านให้ผู้อื่น และควรออกจากระบบทุกครั้งเมื่อใช้งานบนเครื่องส่วนกลาง

## 🔑 เปลี่ยนรหัสผ่าน

ผู้ใช้งานสามารถเปลี่ยนรหัสผ่านของตนเองได้ โดยต้องทราบรหัสผ่านปัจจุบันก่อน

1. คลิกชื่อผู้ใช้งานหรือรูปโปรไฟล์ที่มุมขวาบน
2. เลือก `เปลี่ยนรหัสผ่าน`
3. กรอก `รหัสผ่านปัจจุบัน`
4. กรอก `รหัสผ่านใหม่`
5. กรอก `ยืนยันรหัสผ่านใหม่`
6. ตรวจเงื่อนไข Password Policy และระดับความแข็งแรง
7. กด `เปลี่ยนรหัสผ่าน`
8. ระบบจะออกจากระบบอัตโนมัติ ให้ Login ใหม่ด้วยรหัสผ่านใหม่

### ✅ Checklist รหัสผ่านใหม่

- [ ] อย่างน้อย 8 ตัวอักษร หรือตามที่ระบบกำหนด
- [ ] มีตัวพิมพ์ใหญ่และตัวพิมพ์เล็ก
- [ ] มีตัวเลข
- [ ] มีอักขระพิเศษ เช่น `@`, `#`, `!`
- [ ] ไม่ใช้ username เป็นส่วนหนึ่งของรหัสผ่าน
- [ ] ไม่ซ้ำกับรหัสผ่านเดิม

> 💡 **จำง่าย:** เปลี่ยนสำเร็จแล้วระบบพาไปหน้า Login เป็นพฤติกรรมปกติ เพื่อยกเลิก session เดิมทั้งหมด

## 🪪 ข้อมูลส่วนตัวของฉัน

ข้อมูลส่วนตัวถูกนำไปใช้กับเอกสารใบลา PDF และการติดต่อระหว่างลา

1. คลิกเมนูผู้ใช้งานมุมขวาบน
2. เลือก `ข้อมูลส่วนตัวของฉัน`
3. ตรวจสอบชื่อ-นามสกุล ตำแหน่ง เบอร์โทรศัพท์ อีเมล และที่อยู่ระหว่างลา
4. แก้ไขข้อมูลที่ระบบอนุญาต
5. กด `บันทึกข้อมูล`

> ✅ **ควรตรวจ:** ชื่อ-นามสกุล ตำแหน่ง เบอร์โทรศัพท์ และอีเมลต้องถูกต้องก่อนสร้างคำขอลา

## 📲 เชื่อมต่อ LINE OA

การเชื่อมต่อ LINE ช่วยให้ได้รับแจ้งเตือนเมื่อคำขอถูกส่ง อนุมัติ หรือไม่อนุมัติ

1. เปิดหน้า `ข้อมูลส่วนตัวของฉัน`
2. กด `เชื่อมต่อ LINE`
3. สแกน QR Code หรือใช้รหัสเชื่อมต่อที่ระบบแสดง
4. เพิ่มเพื่อน LINE OA ของ HOP
5. พิมพ์รหัสเชื่อมต่อใน LINE OA
6. ตรวจสอบว่าสถานะแสดงว่า `เชื่อมต่อแล้ว`

> 🔎 **ถ้าไม่ได้รับ LINE:** ตรวจว่าเพิ่มเพื่อน LINE OA แล้ว และบัญชี LINE ไม่ได้ผูกกับผู้ใช้อื่น

## 📝 สร้างคำขอลา

1. ไปที่เมนู `ระบบลา` > `รายการคำขอลา`
2. กด `เพิ่มคำขอลา`
3. เลือกประเภทการลา
4. เลือกรูปแบบการลา เช่น เต็มวัน ครึ่งวันเช้า หรือครึ่งวันบ่าย
5. เลือกวันที่เริ่มต้นและวันที่สิ้นสุด
6. กรอกเหตุผลการลา
7. แนบเอกสารถ้ามีหรือถ้าประเภทลานั้นกำหนดให้แนบ
8. ตรวจสอบจำนวนวันลาและยอดคงเหลือ
9. กด `บันทึก` หรือ `ส่งคำขอ`

### 📎 ไฟล์แนบและการดูตัวอย่าง

1. ในหน้ารายละเอียดคำขอลา ให้ดูส่วน `ไฟล์แนบ`
2. กด `ดูตัวอย่าง` เพื่อเปิดไฟล์ในหน้าต่าง preview
3. รองรับไฟล์ PDF, JPG, JPEG, PNG และ WEBP
4. หากไฟล์ไม่รองรับ ระบบจะแสดงข้อความ `ไม่รองรับการแสดงตัวอย่างไฟล์ประเภทนี้`
5. ปุ่ม `ดาวน์โหลด` ยังใช้ได้ตามสิทธิ์ หากจำเป็นต้องเก็บไฟล์ไว้ตรวจสอบ

> 💡 **Tip:** ถ้าคำขออยู่สถานะ `แบบร่าง` หรือ `ตีกลับรอแก้ไข` ผู้ขอสามารถอัปโหลด ลบ หรือเปลี่ยนไฟล์แนบได้ แต่หลังส่งอนุมัติแล้วจะดูตัวอย่างหรือดาวน์โหลดได้เท่านั้น

### ✅ Checklist ก่อนส่งคำขอ

- [ ] ประเภทลาถูกต้อง
- [ ] วันที่ลาไม่ตรงวันหยุดราชการหรือเสาร์-อาทิตย์
- [ ] รูปแบบเต็มวัน/ครึ่งวันถูกต้อง
- [ ] จำนวนวันที่ระบบคำนวณถูกต้อง
- [ ] ยอดวันลาคงเหลือเพียงพอ
- [ ] แนบเอกสารครบถ้วนถ้าจำเป็น
- [ ] เหตุผลการลาชัดเจนและเหมาะสม

## 📌 ตรวจสอบสถานะคำขอ

ดูได้จาก Dashboard หรือเมนู `รายการคำขอลา`

1. หากอยู่ที่ Dashboard ให้ดู Card `คำขอลาของฉันที่รออนุมัติ` หรือ Card สรุปคำขอของฉัน
2. กด `ดูทั้งหมด` เพื่อไปยังหน้า `รายการคำขอลา`
3. ในหน้า `รายการคำขอลา` เลือกตัวกรอง `ขอบเขตรายการ` เป็น `คำขอของฉัน`
4. เลือกสถานะที่ต้องการ เช่น `รออนุมัติ`, `ตีกลับรอแก้ไข`, `อนุมัติแล้ว`
5. คลิกรายการเพื่อดูรายละเอียดและสายอนุมัติ

สถานะที่พบบ่อย:

| สถานะ | ความหมาย | ควรทำอย่างไร |
|---|---|---|
| Draft | บันทึกไว้แต่ยังไม่ส่งอนุมัติ | ตรวจข้อมูลแล้วกดส่งคำขอ |
| Pending | อยู่ระหว่างรออนุมัติ | รอผู้อนุมัติตามลำดับขั้น |
| ReturnedForRevision | ตีกลับรอแก้ไข | อ่านเหตุผล แก้ไขข้อมูลหรือไฟล์แนบ แล้วกดส่งคำขอใหม่ |
| Approved | อนุมัติเรียบร้อยแล้ว | ดาวน์โหลด PDF เก็บไว้ได้ |
| Rejected | ไม่อนุมัติ | อ่านเหตุผลและสร้างคำขอใหม่ถ้าจำเป็น |
| Cancelled | ยกเลิกคำขอ | ไม่ถูกนำไปคิดเป็นวันลา |

## 🔁 กรณีคำขอถูกตีกลับรอแก้ไข

ถ้าผู้อนุมัติพบว่าข้อมูลหรือเอกสารยังไม่ครบ ระบบจะแสดงสถานะ `ตีกลับรอแก้ไข` พร้อมเหตุผลที่ผู้อนุมัติระบุ

1. เปิดรายละเอียดคำขอลา
2. อ่านเหตุผลการตีกลับ
3. กด `แก้ไขคำขอ` เพื่อแก้ไขข้อมูลที่จำเป็น
4. อัปโหลดหรือลบไฟล์แนบเพิ่มเติม หากผู้อนุมัติร้องขอ
5. ตรวจสอบข้อมูลอีกครั้ง
6. กด `ส่งคำขอใหม่`

> ✅ **จำง่าย:** `ตีกลับรอแก้ไข` ไม่ใช่ `ไม่อนุมัติ` แต่เป็นการส่งกลับให้ผู้ขอปรับข้อมูลก่อนเข้ากระบวนการอนุมัติอีกครั้ง

## 📄 ดาวน์โหลด PDF ใบลา

1. เปิดรายละเอียดคำขอลา
2. กด `ดาวน์โหลดแบบฟอร์มใบลา`
3. ตรวจสอบไฟล์ PDF ที่ดาวน์โหลด

> 🖨️ **หมายเหตุ:** PDF จะแสดงข้อมูลจาก Profile, ประเภทลา, ช่วงเวลา, ยอดวันลา และ Timeline การอนุมัติ

## 🚫 ยกเลิกคำขอ

1. เปิดคำขอลาที่ต้องการยกเลิก
2. ตรวจสอบว่าสถานะยังสามารถยกเลิกได้
3. กด `ยกเลิกคำขอ`
4. ยืนยันการยกเลิก

> ℹ️ **หมายเหตุ:** หากคำขอได้รับอนุมัติครบแล้ว อาจต้องติดต่อ HR หรือผู้ดูแลระบบตามแนวทางของโรงพยาบาล
