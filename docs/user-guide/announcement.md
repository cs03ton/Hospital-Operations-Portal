# 📢 คู่มือศูนย์ข่าวสารและประกาศ

คู่มือนี้อธิบายการใช้งานระบบข่าวสารและประกาศประชาสัมพันธ์ของ Hospital Operations Portal (HOP) สำหรับเจ้าหน้าที่ หัวหน้างาน ผู้บริหาร และผู้ดูแลระบบ

> 💡 **ภาพรวมเร็ว:** อ่านประกาศ → ดูรายละเอียด/รูปภาพ/ไฟล์แนบ → กดรับทราบถ้าระบบกำหนด → ผู้ดูแลสร้างประกาศ กำหนดกลุ่มเป้าหมาย และเลือกช่องทางแจ้งเตือน

## 🧭 เมนูที่เกี่ยวข้อง

| เมนู | ใช้สำหรับ | ผู้ใช้ที่เห็น |
|---|---|---|
| `ข่าวสาร` > `ศูนย์ประกาศ` | อ่านประกาศที่เกี่ยวข้องกับตนเอง | ทุกบทบาทที่มีสิทธิ์ |
| `ข่าวสาร` > `จัดการประกาศ` | สร้าง แก้ไข เผยแพร่ จัดเก็บ ยกเลิก หรือลบประกาศ | Admin / SuperAdmin / ผู้ได้รับสิทธิ์ |
| `Dashboard Hub` | แสดงประกาศสำคัญล่าสุด | ผู้ใช้ที่อยู่ในกลุ่มเป้าหมาย |
| Notification Bell | แจ้งเตือนประกาศผ่านกระดิ่ง | ผู้รับตาม target |
| LINE Official Account | แจ้งเตือนประกาศผ่าน LINE Flex Card | ผู้รับที่เชื่อมต่อ LINE แล้ว |

## 👤 สำหรับผู้ใช้งานทั่วไป

1. ไปที่เมนู `ข่าวสาร` > `ศูนย์ประกาศ`
2. ดูรายการประกาศที่ระบบแสดงให้ตามกลุ่มเป้าหมายของคุณ
3. ใช้ช่องค้นหาเพื่อค้นจากหัวข้อหรือเนื้อหา
4. เลือกตัวกรอง เช่น ความสำคัญ หรือหมวดหมู่ หากมี
5. กด `อ่านรายละเอียด`
6. ตรวจเนื้อหา รูปหน้าปก รูปภาพประกอบ และไฟล์แนบ
7. หากประกาศกำหนดให้รับทราบ ให้กด `รับทราบประกาศ`

> ✅ **จำง่าย:** ผู้ใช้จะเห็นเฉพาะประกาศที่ target ถึงตนเอง เช่น ทุกคน หน่วยงาน บทบาท ผู้ใช้รายบุคคล หรือ permission ที่เกี่ยวข้อง

## 🖼️ รูปภาพและไฟล์แนบในประกาศ

ประกาศรองรับสื่อประกอบหลายรูปแบบ เพื่อให้ข้อมูลอ่านง่ายและตรวจสอบได้ครบถ้วน

| ส่วนประกอบ | รายละเอียด |
|---|---|
| รูปหน้าปก | แสดงใน Card, Dashboard และหน้ารายละเอียด |
| รูปภาพประกอบ | แสดงเป็น Gallery ในหน้ารายละเอียด |
| Preview รูปภาพ | คลิกรูปเพื่อเปิด preview บนหน้าเว็บ ไม่ต้องเปิดหน้าใหม่ |
| ไฟล์แนบ | รองรับเอกสาร เช่น PDF, DOCX, XLSX, PPTX และ ZIP |
| Placeholder | ถ้าไม่มีรูป ระบบใช้รูปแบบแสดงผลมาตรฐานแทน |

> 💡 **Tip:** หากรูปไม่แสดง ให้ลอง refresh หน้าเว็บก่อน หากยังไม่แสดงให้แจ้ง IT พร้อมชื่อประกาศและเวลาที่พบปัญหา

## ✅ การรับทราบประกาศ

ประกาศบางรายการอาจกำหนดให้เจ้าหน้าที่ต้องกดรับทราบ เช่น ประกาศนโยบาย ประกาศเวร หรือประกาศที่มีผลต่อการปฏิบัติงาน

1. เปิดหน้ารายละเอียดประกาศ
2. อ่านเนื้อหาให้ครบถ้วน
3. ตรวจไฟล์แนบ หากมี
4. กด `รับทราบประกาศ`
5. ระบบบันทึกวันเวลาและชื่อผู้รับทราบ

> ⚠️ **ข้อควรระวัง:** การกดรับทราบหมายถึงผู้ใช้ได้อ่านและรับทราบข้อมูลตามประกาศแล้ว ควรอ่านรายละเอียดให้ครบก่อนกด

## 🛠️ สำหรับผู้ดูแลประกาศ

1. ไปที่ `ข่าวสาร` > `จัดการประกาศ`
2. กด `เพิ่มประกาศ`
3. กรอกหัวข้อ สรุป และเนื้อหา
4. เลือกหมวดหมู่และความสำคัญ
5. กำหนดเวลาเผยแพร่และหมดอายุ หากต้องการตั้งเวลา
6. เลือกรูปแบบการแสดงผล เช่น `ประกาศเด่น`, `Popup`, `Banner`, `ต้องรับทราบ`
7. เลือกช่องทางแจ้งเตือน เช่น Notification Bell หรือ LINE Official Account
8. เลือกกลุ่มเป้าหมาย
9. บันทึกข้อมูล
10. อัปโหลดรูปหน้าปก รูปภาพประกอบ หรือไฟล์แนบ หากมี
11. ตรวจ preview ก่อนเผยแพร่
12. กด `เผยแพร่`

## 🎯 การเลือกกลุ่มเป้าหมาย

ระบบรองรับการเลือก target หลายรูปแบบ และสามารถเลือกค่าได้หลายรายการตามประเภท target

| กลุ่มเป้าหมาย | ตัวอย่างการใช้งาน |
|---|---|
| ทุกคน | ประกาศแจ้งปิดระบบ หรือข่าวประชาสัมพันธ์ทั้งโรงพยาบาล |
| หน่วยงาน | เลือกหลายหน่วยงาน เช่น IT, HR, งานการเงิน |
| บทบาท | เลือก Staff, Head, Director, Admin หรือบทบาทอื่น |
| บุคคล | ค้นหาและเลือกผู้ใช้หลายคนจากรายชื่อผู้ใช้ในระบบ |
| Permission | ส่งถึงผู้ใช้ที่มีสิทธิ์เฉพาะ เช่น ผู้ดูแลระบบหรือผู้อนุมัติ |

> 💡 **Tip:** ก่อนเผยแพร่ให้ตรวจจำนวนผู้รับใน preview โดยเฉพาะกรณีส่ง LINE เพราะผู้รับบางคนอาจยังไม่ได้เชื่อมต่อ LINE

## 🔔 ช่องทางแจ้งเตือนประกาศ

| ช่องทาง | ผลลัพธ์ |
|---|---|
| ไม่เลือกช่องทาง | ประกาศยังแสดงในศูนย์ประกาศและ Dashboard แต่ไม่ส่งแจ้งเตือนเพิ่ม |
| Notification Bell | สร้างรายการแจ้งเตือนในระบบ |
| LINE Official Account | ส่ง LINE Flex Card พร้อมปุ่ม `ดูรายละเอียด` |
| ทั้งสองช่องทาง | ส่งทั้ง Notification Bell และ LINE |

> ⚠️ **ข้อควรระวัง:** LINE เป็น best-effort channel หาก LINE API ล่ม หรือผู้ใช้ยังไม่เชื่อมต่อ LINE ประกาศยังเผยแพร่ได้ตามปกติ แต่รายการส่ง LINE อาจอยู่ในสถานะรอส่งหรือล้มเหลว

## 📌 สถานะประกาศ

| สถานะ | ความหมาย | การดำเนินการที่พบบ่อย |
|---|---|---|
| แบบร่าง | ยังไม่เผยแพร่ | แก้ไข อัปโหลดรูป/ไฟล์ ลบ |
| ตั้งเวลา | รอถึงเวลาที่กำหนด | แก้ไข ยกเลิก หรือตรวจ preview |
| เผยแพร่แล้ว | ผู้ใช้กลุ่มเป้าหมายเห็นประกาศแล้ว | ดูรายละเอียด จัดเก็บ ยกเลิก ทำสำเนา |
| หมดอายุ | เลยเวลาสิ้นสุด | ตรวจประวัติหรือทำสำเนา |
| จัดเก็บแล้ว | ปิดประกาศเพื่อเก็บประวัติ | ดูย้อนหลัง |
| ยกเลิก | ประกาศถูกยกเลิก | ตรวจ audit หรือทำสำเนา |

## 🗑️ การลบประกาศ

ผู้ดูแลที่มีสิทธิ์สามารถลบประกาศจากหน้าจัดการประกาศได้

1. เปิด `ข่าวสาร` > `จัดการประกาศ`
2. เลือกรายการประกาศ
3. กดปุ่มลบ
4. ตรวจข้อความยืนยัน
5. ยืนยันการลบ

> ⚠️ **Warning:** การลบจะลบ metadata และสื่อประกอบที่เกี่ยวข้องตาม workflow ของระบบ ควรใช้กับประกาศที่ไม่จำเป็นต้องเก็บประวัติเท่านั้น หากต้องการเก็บประวัติให้ใช้ `จัดเก็บ` แทน

## 📊 ข้อมูลที่ระบบติดตาม

ระบบเก็บข้อมูลเพื่อช่วยให้ผู้ดูแลตรวจสอบผลการสื่อสารได้

- จำนวนผู้เข้าชม
- จำนวนผู้รับทราบ
- กลุ่มเป้าหมายของประกาศ
- ช่องทางแจ้งเตือนที่เลือก
- สถานะการส่ง Notification Bell
- สถานะการส่ง LINE
- Audit Log เช่น สร้าง แก้ไข เผยแพร่ ยกเลิก จัดเก็บ ลบ

## ✅ Checklist ก่อนเผยแพร่ประกาศ

- [ ] หัวข้อชัดเจนและเหมาะกับผู้รับ
- [ ] สรุปสั้น เข้าใจง่าย
- [ ] เนื้อหาครบถ้วน ไม่มีข้อมูลส่วนบุคคลเกินจำเป็น
- [ ] เลือกหมวดหมู่และความสำคัญถูกต้อง
- [ ] เลือกกลุ่มเป้าหมายถูกต้อง
- [ ] ตรวจรูปหน้าปกและรูปภาพประกอบ
- [ ] ตรวจไฟล์แนบว่าเปิดได้
- [ ] เลือกช่องทางแจ้งเตือนถูกต้อง
- [ ] ตรวจ preview จำนวนผู้รับก่อนเผยแพร่
- [ ] ตรวจว่าประกาศที่ต้องรับทราบเปิดตัวเลือก `ต้องรับทราบ`

## 🧯 Troubleshooting

| ปัญหา | วิธีตรวจเบื้องต้น |
|---|---|
| ไม่เห็นประกาศ | ตรวจกลุ่มเป้าหมาย สถานะเผยแพร่ และวันหมดอายุ |
| กดรูปแล้วไม่ preview | refresh หน้าเว็บ และตรวจว่าไฟล์เป็นรูปภาพที่รองรับ |
| ดาวน์โหลดไฟล์แนบไม่ได้ | ตรวจสิทธิ์ประกาศ และแจ้ง IT พร้อมชื่อไฟล์ |
| LINE ไม่ได้รับ | ตรวจว่าผู้ใช้เชื่อมต่อ LINE แล้ว และประกาศเลือกช่องทาง LINE |
| Notification Bell ไม่ขึ้น | ตรวจว่าเลือกช่องทาง Notification Bell และผู้ใช้อยู่ใน target |
| ปุ่มรับทราบไม่แสดง | ประกาศนั้นอาจไม่ได้กำหนด `ต้องรับทราบ` |

เอกสารนี้เป็นส่วนหนึ่งของโครงการ Hospital Operations Portal (HOP) โรงพยาบาลนาหมื่น
