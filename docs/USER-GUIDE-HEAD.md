# คู่มือหัวหน้าหน่วยงาน

เอกสารหลักอยู่ที่ [`docs/user-guide/head.md`](user-guide/head.md)

## Dashboard ล่าสุด

Dashboard สำหรับหัวหน้าหน่วยงานแยกข้อมูลเป็น 2 ส่วน:

1. `คำขอลาของฉันที่รออนุมัติ`
   - แสดงเฉพาะคำขอลาของหัวหน้าเองที่อยู่ในสถานะ `รออนุมัติ`
   - กด `ดูทั้งหมด` เพื่อไปที่ `/leave?scope=mine&status=pending`
2. `คำขอลาของหน่วยงาน`
   - แสดงคำขอของเจ้าหน้าที่ในหน่วยงานเดียวกัน
   - ไม่รวมคำขอของหัวหน้าเอง
   - กด `ดูทั้งหมด` เพื่อไปที่ `/leave?scope=department`

> เอกสารนี้เป็นไฟล์อ้างอิงสั้นสำหรับ requirement เฉพาะ ส่วนคู่มือใช้งานฉบับเต็มให้แก้ไขและอ่านจาก `docs/user-guide/head.md`
