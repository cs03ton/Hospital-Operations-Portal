# คู่มือระบบลา: การแก้ไขและไฟล์แนบ

## แบบร่าง

เมื่อคำขอยังเป็นแบบร่าง ผู้ขอสามารถแก้ไขข้อมูล อัปโหลดไฟล์แนบ ลบไฟล์แนบ และส่งคำขอได้

## หลังส่งอนุมัติ

เมื่อส่งคำขอแล้ว ผู้ขอจะดูข้อมูล ดูตัวอย่างไฟล์แนบ และดาวน์โหลดไฟล์แนบได้เท่านั้น ไม่สามารถแก้ไขหรือลบไฟล์แนบได้

## ติดตามคำขอจาก Dashboard และรายการคำขอลา

ผู้ใช้งานทั่วไปและหัวหน้าหน่วยงานสามารถติดตามคำขอของตนเองจาก Dashboard ได้

- Card `คำขอลาของฉันที่รออนุมัติ` เปิดรายการคำขอลาที่เป็นของตนเองและอยู่ระหว่างอนุมัติ
- หน้า `รายการคำขอลา` มีตัวกรอง `ขอบเขตรายการ`
- เลือก `คำขอของฉัน` เพื่อดูเฉพาะคำขอของตนเอง
- เลือกสถานะ เช่น `รออนุมัติ`, `ตีกลับรอแก้ไข`, `อนุมัติแล้ว`, `ไม่อนุมัติ`, `ยกเลิกแล้ว`

สำหรับหัวหน้าหน่วยงาน ระบบมี Card แยกต่างหากชื่อ `คำขอลาของหน่วยงาน` เพื่อดูรายการของเจ้าหน้าที่ในหน่วยงานเดียวกัน โดยไม่รวมคำขอของหัวหน้าเอง

Dashboard ล่าสุดจะแสดง `คำขอยกเลิกใบลา` เป็น Card แยกจากคำขอลาปกติ เพื่อให้ไม่ปะปนจำนวนคำขอลาและจำนวนวันที่คืนยอด:

- Staff เห็นคำขอยกเลิกใบลาของตนเอง
- Department Head เห็นคำขอของตนเองและข้อมูลหน่วยงานตามสิทธิ์
- Director เห็นข้อมูลส่วนตัวก่อน แล้วตามด้วยข้อมูลเชิงบริหาร
- Admin/SuperAdmin เห็นข้อมูลเพื่อ monitor/support ตามสิทธิ์ แต่ไม่ได้รับสิทธิ์สร้างคำขอลาโดย default

## ดูตัวอย่างไฟล์แนบ

1. เปิดรายละเอียดคำขอลา
2. ไปที่ส่วน `ไฟล์แนบ`
3. กด `ดูตัวอย่าง`
4. ระบบจะแสดง PDF หรือรูปภาพในหน้าต่าง Preview

รองรับไฟล์ PDF, JPG/JPEG, PNG และ WEBP หากไฟล์ไม่รองรับ ระบบจะแสดงข้อความ `ไม่รองรับการแสดงตัวอย่างไฟล์ประเภทนี้`

## ตีกลับรอแก้ไข

ถ้าผู้อนุมัติตีกลับคำขอ ระบบจะแสดงเหตุผลให้เห็นชัดเจน ผู้ขอสามารถ:

1. แก้ไขข้อมูลคำขอ
2. อัปโหลดหรือลบไฟล์แนบ
3. ส่งคำขอใหม่
4. ยกเลิกคำขอ ถ้าไม่ต้องการดำเนินการต่อ

ข้อความที่จะแสดงในระบบคือ “ตีกลับรอแก้ไข” หมายถึงคำขอยังไม่ถูกปฏิเสธถาวร แต่รอให้ผู้ขอแก้ไขข้อมูลหรือไฟล์แนบแล้วส่งใหม่

## ข้อควรระวัง

หลังคำขออนุมัติครบแล้ว ไม่สามารถแก้ไขข้อมูลหรือไฟล์แนบได้

## การขอยกเลิกใบลา

เมื่อใบลาของท่านได้รับอนุมัติแล้ว แต่ภายหลังต้องการยกเลิกใบลาทั้งรายการ ให้ดำเนินการดังนี้:

1. เปิดเมนู `ระบบลา > คำขอยกเลิกใบลา`
2. กด `เลือกใบลาที่ต้องการยกเลิก`
3. เลือกใบลาเดิมที่มีสถานะ `อนุมัติแล้ว`
4. ตรวจสอบเลขที่คำขอเดิม ประเภทลา วันที่ลา และจำนวนวันที่จะคืนยอด
5. ระบุเหตุผล
6. กด `ส่งคำขอ`

> **หมายเหตุ:** ระบบจะเปลี่ยนสถานะใบลาเดิมและคืนยอดวันลาให้หลังจากคำขอยกเลิกใบลาได้รับอนุมัติครบทุกขั้นแล้วเท่านั้น

สามารถติดตามคำขอยกเลิกใบลาได้จากเมนู `ระบบลา > คำขอยกเลิกใบลา`

หน้า `คำขอยกเลิกใบลา` รองรับตัวกรองและ paging เช่นเดียวกับหน้า `รายการคำขอลา`:

| ตัวกรอง | ใช้ทำอะไร |
|---|---|
| ประเภทลา | ค้นหาตามประเภทของใบลาเดิม |
| สถานะคำขอ | ดูรายการรออนุมัติ อนุมัติแล้ว ไม่อนุมัติ ตีกลับ หรือยกเลิก |
| ขอบเขตรายการ | ดูคำขอของตนเอง หรือรายการตามสิทธิ์ |
| ผู้ขอลา | ค้นหาตามผู้ขอเมื่อมีสิทธิ์ดูหลายคน |
| ตั้งแต่วันที่ / ถึงวันที่ | ค้นหาตามช่วงวันที่ของใบลาเดิม |

ใช้ตัวเลือกจำนวนรายการต่อหน้าและปุ่มเปลี่ยนหน้าเมื่อมีหลายรายการ

หากใบลาเดิมถูกยกเลิกหลังอนุมัติแล้ว หน้ารายละเอียดใบลาเดิมจะแสดง banner พร้อมเลขที่คำขอยกเลิกใบลา `LVC-xxxx`
