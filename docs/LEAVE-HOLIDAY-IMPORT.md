# Leave Holiday Import

ระบบรองรับการนำเข้าวันหยุดราชการหลายปีผ่านไฟล์ `.csv` หรือ `.xlsx`

## Template

ดาวน์โหลด template จากหน้า `วันหยุดราชการ` หรือ API:

```http
GET /api/leave-holidays/import-template
```

คอลัมน์ที่ต้องมี:

```text
วันที่,ชื่อวันหยุด,ประเภทวันหยุด
2027-01-01,วันขึ้นปีใหม่,National
2027-04-06,วันจักรี,National
2027-04-13,วันสงกรานต์,National
```

## Date Format

วันที่ในไฟล์ต้องใช้:

```text
YYYY-MM-DD
```

ตัวอย่าง:

```text
2027-01-01
```

ในหน้าเว็บระบบจะแสดงวันที่เป็นรูปแบบไทย เช่น `01/01/2570`

## Import Flow

1. กด `Import วันหยุดราชการ`
2. เลือกไฟล์ `.csv` หรือ `.xlsx`
3. กด `Preview ก่อน Import`
4. ตรวจผล validation
5. กด `Confirm Import`

หลัง import เสร็จ ตารางจะ refresh ตาม filter ปีปัจจุบัน หากนำเข้าปีอื่น ให้เลือก filter ปีนั้นเพื่อดูข้อมูล

## Validation

ระบบตรวจ:

- วันที่ต้องถูกต้อง
- ชื่อวันหยุดต้องไม่ว่าง
- วันที่ซ้ำ
- ชื่อวันหยุดซ้ำ
- ปีต้องไม่เก่าเกิน policy ปัจจุบัน

ถ้ามีแถวผิดพลาด ระบบจะไม่ import และจะแสดงเหตุผลใน preview
