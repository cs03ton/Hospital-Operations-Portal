# ประเภทการลา

HOP Phase 1 ใช้รหัสประเภทลามาตรฐานดังนี้

| Code | ภาษาไทย | ใช้ยอดคงเหลือ | ใช้ปีงบประมาณ | สะสมได้ |
|---|---|---|---|---|
| `VACATION_LEAVE` | ลาพักผ่อน | ใช่ | ใช่ | ตาม policy |
| `PERSONAL_LEAVE` | ลากิจส่วนตัว | ใช่ | ใช่ | ไม่ |
| `SICK_LEAVE` | ลาป่วย | ใช่ | ใช่ | ไม่ |
| `MATERNITY_LEAVE` | ลาคลอดบุตร | ไม่ | ใช่ | ไม่ |
| `ORDINATION_LEAVE` | ลาบวช | ไม่ | ไม่ | ไม่ |

ระบบยังรองรับประเภทอื่นที่มีอยู่เดิม เช่น ลาศึกษาต่อ และอื่น ๆ โดยสามารถตั้งค่า `requiresBalance = false` ได้

## Legacy Code

ข้อมูลเดิมที่ใช้ code เช่น `AnnualLeave`, `SickLeave`, `PersonalLeave` จะถูก seed/normalize ไปยัง code มาตรฐานใหม่เพื่อหลีกเลี่ยงประเภทลาซ้ำในหน้าจัดการประเภทลา
