# คู่มือบัญชีผู้ใช้งาน

คู่มือนี้อธิบายการจัดการบัญชีผู้ใช้งานส่วนตัวในระบบ Hospital Operations Portal (HOP)

## เปลี่ยนรหัสผ่าน

1. เข้าสู่ระบบ HOP
2. กดชื่อผู้ใช้งานหรือรูปโปรไฟล์มุมขวาบน
3. เลือก `เปลี่ยนรหัสผ่าน`
4. กรอก `รหัสผ่านปัจจุบัน`
5. กรอก `รหัสผ่านใหม่`
6. กรอก `ยืนยันรหัสผ่านใหม่`
7. ตรวจสอบระดับความแข็งแรงของรหัสผ่าน
8. กด `เปลี่ยนรหัสผ่าน`
9. ระบบจะแจ้งว่าเปลี่ยนรหัสผ่านสำเร็จ และนำกลับไปหน้าเข้าสู่ระบบ
10. เข้าสู่ระบบใหม่ด้วยรหัสผ่านใหม่

> **Note:** หลังเปลี่ยนรหัสผ่าน ระบบจะออกจากระบบทุก session เพื่อความปลอดภัย

## เงื่อนไขรหัสผ่าน

- [ ] มีความยาวอย่างน้อย 8 ตัวอักษร หรือตาม policy ที่ผู้ดูแลระบบกำหนด
- [ ] มีตัวพิมพ์ใหญ่อย่างน้อย 1 ตัว
- [ ] มีตัวพิมพ์เล็กอย่างน้อย 1 ตัว
- [ ] มีตัวเลขอย่างน้อย 1 ตัว
- [ ] มีอักขระพิเศษอย่างน้อย 1 ตัว
- [ ] ไม่ใช้ username เป็นส่วนหนึ่งของรหัสผ่าน
- [ ] ไม่ซ้ำกับรหัสผ่านเดิม

## กรณีลืมรหัสผ่าน

หากจำรหัสผ่านปัจจุบันไม่ได้ ให้ติดต่อผู้ดูแลระบบเพื่อ reset password ผ่าน User Management

> **Warning:** อย่าส่งรหัสผ่านผ่าน LINE, อีเมล หรือช่องทางสนทนาสาธารณะ

## สิ่งที่ควรแจ้ง IT เมื่อพบปัญหา

1. Username
2. เวลาที่พบปัญหา
3. ข้อความ error ที่ระบบแสดง
4. หน้าที่ใช้งาน เช่น `เปลี่ยนรหัสผ่าน`

เอกสารนี้เป็นส่วนหนึ่งของโครงการ Hospital Operations Portal (HOP) โรงพยาบาลนาหมื่น
