# User Guide: Leave Request

คู่มือย่อสำหรับเจ้าหน้าที่ในการสร้างคำขอลา

## สร้างคำขอลา

1. เข้าเมนู `ระบบลา`
2. เลือก `รายการคำขอลา`
3. กด `เพิ่มคำขอลา`
4. เลือกประเภทการลา
5. เลือกประเภทช่วงเวลา
   - เต็มวัน
   - ครึ่งวัน (เช้า)
   - ครึ่งวัน (บ่าย)
6. เลือกวันที่จาก DatePicker
7. กรอกเหตุผล
8. กด `บันทึกแบบร่าง`
9. ตรวจรายละเอียด แล้วกด `ส่งคำขอ`

## ไฟล์แนบ

ผู้ขอสามารถอัปโหลด ลบ และเปลี่ยนไฟล์แนบได้ในสถานะ:

- `แบบร่าง`
- `ตีกลับรอแก้ไข`

หลังส่งคำขอแล้ว ผู้ขอและผู้อนุมัติจะใช้ปุ่ม `ดูตัวอย่าง` เป็นหลัก และดาวน์โหลดได้ตามสิทธิ์ที่ระบบอนุญาต

ไฟล์ที่รองรับการดูตัวอย่าง:

- PDF
- JPG/JPEG
- PNG
- WEBP

## ยอดวันลาคงเหลือ

เมื่อเลือกประเภทการลา ระบบจะแสดง:

- คงเหลือกี่วัน
- คำขอนี้ใช้ประมาณกี่วัน
- มียอดรออนุมัติอยู่กี่วัน

ถ้ายอดไม่พอ ระบบจะแสดงคำเตือน:

```text
ยอดวันลาไม่เพียงพอ
```

และ backend จะไม่อนุญาตให้ส่งคำขอเกินสิทธิ์

## การลาครึ่งวัน

ถ้าเลือกครึ่งวัน ระบบจะใช้วันที่เดียวกันสำหรับวันที่เริ่มลาและวันที่สิ้นสุด และคิดเป็น `0.5` วัน

## หลังส่งคำขอ

เมื่อส่งคำขอแล้ว ระบบจะกันยอดไว้ใน `รออนุมัติ`

- ถ้าอนุมัติครบ ยอดจะถูกย้ายไป `ใช้ไป`
- ถ้าไม่อนุมัติหรือยกเลิก ยอดที่กันไว้จะถูกคืน

## การดูรายการคำขอลาของฉัน

ผู้ใช้งานสามารถติดตามคำขอของตนเองได้จากเมนู `รายการคำขอลา` หรือจาก Dashboard

1. เข้าเมนู `ระบบลา`
2. เลือก `รายการคำขอลา`
3. ใช้ตัวกรอง `ขอบเขตรายการ`
4. เลือก `คำขอของฉัน`
5. เลือกสถานะที่ต้องการ เช่น `รออนุมัติ`, `อนุมัติแล้ว`, `ไม่อนุมัติ`, `ตีกลับรอแก้ไข`

ลิงก์จาก Dashboard สำหรับคำขอของตนเองที่รออนุมัติจะเปิดรายการด้วยตัวกรอง:

```text
/leave?scope=mine&status=pending
```

> **Note:** ระบบจะแสดงเฉพาะข้อมูลตามสิทธิ์ของผู้ใช้งาน ถึงแม้จะเปลี่ยน URL เอง backend ก็ยังตรวจสิทธิ์ซ้ำเสมอ

## ตีกลับรอแก้ไข

ถ้าผู้อนุมัติส่งคำขอกลับมา ระบบจะแสดงสถานะ `ตีกลับรอแก้ไข`

1. เปิดรายละเอียดคำขอ
2. อ่านเหตุผลที่ผู้อนุมัติระบุ
3. แก้ไขข้อมูลหรือไฟล์แนบ
4. กด `ส่งคำขอใหม่`

ผู้ใช้อื่น ผู้อนุมัติ Admin หรือ SuperAdmin ไม่สามารถส่งคำขอใหม่แทนผู้ขอได้
