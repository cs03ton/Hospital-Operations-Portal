# คู่มือผู้อนุมัติ: ตีกลับรอแก้ไข

## ใช้เมื่อใด

ใช้ “ตีกลับรอแก้ไข” เมื่อข้อมูลคำขอลาหรือเอกสารแนบยังไม่ครบ แต่ยังไม่ต้องการไม่อนุมัติคำขอ

## ขั้นตอน

1. เปิดรายละเอียดคำขอลา
2. ตรวจสอบข้อมูลและไฟล์แนบ โดยกด `ดูตัวอย่าง` ก่อนดาวน์โหลด
3. กด “ตีกลับรอแก้ไข”
4. ระบุเหตุผล เช่น “กรุณาแนบใบรับรองแพทย์”
5. ยืนยันการตีกลับ

หลังตีกลับแล้ว ผู้อนุมัติจะไม่สามารถอนุมัติคำขอนั้นต่อได้จนกว่าผู้ขอจะส่งใหม่

เมื่อผู้ขอแก้ไขและกด `ส่งคำขอใหม่` ระบบจะแจ้งกลับมายังผู้อนุมัติ step เดิม

## การอนุมัติคำขอยกเลิกใบลา

คำขอยกเลิกใบลาเป็นเอกสารแยกจากใบลาเดิม ผู้อนุมัติจะเห็นเฉพาะรายการที่ถึงคิวตนเองเหมือนการอนุมัติใบลาปกติ

หน้ารายละเอียดคำขอยกเลิกใบลาจะแสดงส่วน `การอนุมัติ` เป็น card/timeline คล้ายหน้ารายละเอียดคำขอลา โดยแสดงลำดับขั้น ผู้อนุมัติ สถานะ วันที่ดำเนินการ และหมายเหตุ

ขั้นตอน:

1. เปิดเมนู `ระบบลา > คำขอยกเลิกใบลา`
2. เปิดรายละเอียดคำขอ
3. ตรวจสอบใบลาเดิม จำนวนวันลา และเหตุผลการยกเลิก
4. เลือก `อนุมัติ`, `ไม่อนุมัติ` หรือ `ตีกลับรอแก้ไข`
5. ใส่หมายเหตุหรือเหตุผลตามความเหมาะสม

> **ข้อควรระวัง:** ใบลาเดิมจะเปลี่ยนสถานะและคืนยอดวันลาเฉพาะหลังอนุมัติครบทุกขั้นเท่านั้น

หากต้องค้นหารายการจำนวนมาก ให้ใช้ตัวกรอง `ประเภทลา`, `สถานะคำขอ`, `ขอบเขตรายการ`, `ผู้ขอลา`, `ตั้งแต่วันที่` และ `ถึงวันที่` ในหน้า `คำขอยกเลิกใบลา`
