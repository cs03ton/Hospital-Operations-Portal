# คู่มือศูนย์ข่าวสารและประกาศ

> เอกสารฉบับเต็มสำหรับอ่านในระบบอยู่ที่ `docs/user-guide/announcement.md` และแสดงใน Documentation Center ด้วย slug `announcement-guide`

## สำหรับเจ้าหน้าที่

1. เข้าเมนู “ข่าวสาร”
2. เลือก “ศูนย์ประกาศ”
3. อ่านรายการประกาศที่เกี่ยวข้องกับตนเอง
4. กด “อ่านรายละเอียด”
5. ดูรูปหน้าปก รูปภาพประกอบ และไฟล์แนบ
6. คลิกรูปภาพประกอบเพื่อ preview บนหน้าเว็บ
7. ถ้าประกาศต้องรับทราบ ให้กด “รับทราบประกาศ”

> **Note:** ผู้ใช้งานจะเห็นเฉพาะประกาศที่ระบบกำหนดกลุ่มเป้าหมายถึงตนเองเท่านั้น

## สำหรับผู้ดูแลระบบ

1. เข้าเมนู “ข่าวสาร”
2. เลือก “จัดการประกาศ”
3. กด “เพิ่มประกาศ”
4. กรอกหัวข้อ สรุป และเนื้อหา
5. เลือกระดับความสำคัญ
6. เลือกกลุ่มเป้าหมาย
7. เลือกค่าเป้าหมายตามประเภท เช่น หลายหน่วยงาน หลายบทบาท หรือหลายบุคคล
8. เลือกรูปแบบการแสดงผล เช่น ประกาศเด่น, Popup, Banner และต้องรับทราบ
9. เลือกช่องทางแจ้งเตือน เช่น Notification Bell, LINE หรือทั้งสองช่องทาง
10. กด “บันทึกข้อมูล”
11. อัปโหลดรูปหน้าปก รูปภาพประกอบ หรือไฟล์แนบ
12. ตรวจ preview จำนวนผู้รับก่อนเผยแพร่
13. กด “เผยแพร่” เมื่อพร้อมใช้งาน

> **Note:** LINE notification จะส่งเฉพาะผู้ใช้ที่เชื่อมต่อ LINE กับบัญชี HOP แล้ว หากส่ง LINE ไม่สำเร็จ ประกาศยังเผยแพร่ได้ตามปกติ

## รูปภาพและไฟล์แนบ

| ประเภท | รายละเอียด |
| --- | --- |
| รูปหน้าปก | ใช้แสดงบน Card, Dashboard และหน้ารายละเอียด |
| รูปภาพประกอบ | แสดงเป็น Gallery และคลิกเพื่อ preview ได้ |
| ไฟล์แนบ | รองรับเอกสารสำนักงานและ ZIP ตาม allowlist |
| Drag & Drop | หน้าจัดการประกาศแสดงพื้นที่ลากไฟล์วางและคำอธิบายชนิดไฟล์ที่รองรับ |

## กลุ่มเป้าหมาย

| ประเภท target | ตัวอย่าง |
| --- | --- |
| ทุกคน | ประกาศทั่วโรงพยาบาล |
| หน่วยงาน | เลือกหลายหน่วยงาน |
| บทบาท | เลือก Staff, Head, Director, Admin หรือ SuperAdmin |
| บุคคล | ค้นหาและเลือกผู้ใช้หลายคน |
| Permission | ส่งถึงผู้ใช้ที่มีสิทธิ์เฉพาะ |

## สถานะประกาศ

| สถานะ | ความหมาย |
| --- | --- |
| แบบร่าง | ยังไม่เผยแพร่ |
| ตั้งเวลา | รอถึงเวลาที่กำหนด |
| เผยแพร่แล้ว | ผู้ใช้กลุ่มเป้าหมายสามารถเห็นได้ |
| หมดอายุ | เลยเวลาสิ้นสุด |
| จัดเก็บแล้ว | ปิดรายการเพื่อเก็บประวัติ |
| ยกเลิก | ประกาศถูกยกเลิก |

## การลบประกาศ

ผู้ดูแลที่มีสิทธิ์สามารถลบประกาศจากหน้าจัดการประกาศได้ ระบบจะลบข้อมูลประกาศและ media ที่เกี่ยวข้องตาม workflow ของ backend

> **Warning:** ถ้าต้องการเก็บประวัติไว้ตรวจสอบ ควรใช้ “จัดเก็บ” แทน “ลบ”

เอกสารนี้เป็นส่วนหนึ่งของโครงการ Hospital Operations Portal (HOP) โรงพยาบาลนาหมื่น
