# ประเภทพนักงานในระบบ HOP

ระบบใช้ `employmentType` เพื่อคำนวณสิทธิ์วันลาตามประเภทการจ้างงานของผู้ใช้งาน

| Code | ภาษาไทย |
|---|---|
| `CIVIL_SERVANT` | ข้าราชการ |
| `GOVERNMENT_EMPLOYEE` | พนักงานราชการ |
| `PERMANENT_EMPLOYEE` | ลูกจ้างประจำ |
| `MOPH_EMPLOYEE` | พนักงานกระทรวงสาธารณสุข |
| `TEMPORARY_EMPLOYEE_MONTHLY` | ลูกจ้างชั่วคราวรายเดือน |
| `TEMPORARY_EMPLOYEE_DAILY` | ลูกจ้างชั่วคราวรายวัน |

## การตั้งค่า

ผู้ดูแลระบบกำหนดได้ที่หน้าเพิ่ม/แก้ไขผู้ใช้งาน:

- ประเภทพนักงาน
- วันที่เริ่มปฏิบัติงาน

ผู้ใช้งานทั่วไปเห็นข้อมูลนี้ในหน้า “ข้อมูลส่วนตัวของฉัน” แบบอ่านอย่างเดียว

## ผลต่อการลา

ถ้าผู้ใช้ยังไม่ได้กำหนด `employmentType` ระบบจะไม่ให้ส่งคำขอลา และแจ้งว่า:

```text
ยังไม่ได้กำหนดประเภทพนักงาน กรุณาติดต่อผู้ดูแลระบบ
```

ถ้า policy กำหนดเงื่อนไขอายุงาน แต่ยังไม่มีวันที่เริ่มปฏิบัติงาน ระบบจะแจ้งให้ติดต่อผู้ดูแลระบบ

## หมายเหตุเรื่องอายุงาน

ระบบไม่ได้ใช้กฎ "ครบ 6 เดือนจึงลาทุกประเภทได้" แบบรวมทั้งระบบ เงื่อนไขอายุงานจะขึ้นกับประเภทลาและประเภทพนักงาน เช่น:

- ลาพักผ่อนต้องมีอายุงานอย่างน้อย 6 เดือน
- ลาป่วยของลูกจ้างชั่วคราวรายเดือนใช้สิทธิ์ปีแรกได้ตาม policy โดยไม่ถูกบล็อกจากกฎ 6 เดือน
- ลากิจส่วนตัวและลาบวชบางประเภทอาจมีเงื่อนไขอายุงานเฉพาะของตัวเอง
