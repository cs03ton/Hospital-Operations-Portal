# Leave Approval

## Normal Approval

ผู้อนุมัติสามารถอนุมัติหรือไม่อนุมัติได้เฉพาะคำขอที่ `current_approver_id` เป็นตนเอง และ approval step ปัจจุบันเป็น `Pending`

## Return for Revision

ปุ่ม “ตีกลับรอแก้ไข” ใช้เมื่อคำขอต้องการข้อมูลหรือเอกสารเพิ่มเติม โดยไม่ปฏิเสธคำขอถาวร

เงื่อนไข:

1. ต้องเป็น current approver
2. ต้องระบุเหตุผล
3. ผู้ขอไม่สามารถตีกลับคำขอของตนเอง
4. คำขอต้องอยู่ในสถานะ `Pending`

เมื่อผู้ขอแก้ไขและส่งใหม่ ระบบจะกลับมาที่ step เดิมที่ตีกลับ

## Leave Cancellation Approval

คำขอยกเลิกใบลาใช้ approval chain เดิมของระบบลา แต่เป็นเอกสารแยกจากใบลาเดิม

เงื่อนไข:

1. ใบลาเดิมต้องอนุมัติแล้ว
2. ผู้ขอต้องเป็นเจ้าของใบลาเดิม
3. ผู้อนุมัติต้องเป็น current approver ของคำขอยกเลิกใบลา
4. ผู้อนุมัติไม่สามารถอนุมัติคำขอของตนเอง
5. ใบลาเดิมจะเปลี่ยนเป็น `CancelledAfterApproval` หลัง final approval เท่านั้น
